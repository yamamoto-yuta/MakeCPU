# MakeCPU

『作ろう！CPU』に従って，CPU(TD4)の自作を行う．
 
- [サポートページ](https://amane-uehara.github.io/cpubook/)

## 環境構築

コンテナの立ち上げ
```
$ git clone https://github.com/yamamoto-yuta/MakeCPU.git
$ docker-compose up
```

クライアント接続
- ホストIPの調べ方：
```
$ hostname -I
```

Vivado起動方法
```
$ <INSTALLDIR>/Vivado/2020.1/bin/vivado
```
